<h1>开始项目</h1>

API 接口初始化框架,拉下即用

获取第三方包
```bash
 go get
 ```
<br>
在GoLand设置中需要勾选 Go mod 启用,不然会有包会红线

对swagger 支持还未完成
安装swagger
```bash
go get -u github.com/swaggo/swag/cmd/swag
```
